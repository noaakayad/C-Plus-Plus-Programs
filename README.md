# C++ Data Structures and Algorithms

This repository contains C++ coursework focused on data structures, algorithms and systems-level programming.

It includes implementations of classic abstract data types, tree structures, probabilistic data structures, suffix arrays and compression-related algorithms.

## Highlights

- Queue implementation.
- Set ADTs implemented with arrays, linked lists, hash tables, tries and red-black trees.
- Binary Search Tree and AVL Tree implementations.
- Bloom Filter and Count-Min Sketch.
- Suffix Array.
- Burrows-Wheeler Transform.
- File compression and decompression with Huffman Trees.
